# Mycosis — Brutalist Personal Blog

A mock personal blog built entirely from scratch in 2020, written from memory without referencing documentation, Stack Overflow, or any external resources.

A deliberate exercise in design confidence and retained knowledge.

---

## About the project

Mycosis is a personal blog concept built around brutalist web design principles — raw structure, monospace type, minimal decoration, and honest use of the browser as a medium.

The name and aesthetic were chosen intentionally. Brutalism in web design strips away the polish that frameworks and templates provide, forcing every decision to be deliberate. There's nowhere to hide behind a component library when everything is written by hand.

---

## What I built

- Full blog layout with a homepage, article grid, and navigation
- Brutalist aesthetic: monospace typography, flat colour, minimal border radius, stark layout
- Written entirely from memory — no documentation lookups, no Stack Overflow, no copy-pasting
- Clean two-column article grid with image and heading structure
- Minimal navigation with a simple pipe-separated style true to the brutalist ethos

---

## The constraint

The point of this project was to find out what I actually knew without a safety net.

No searching. No referencing. Just HTML and CSS written from what was already in my head.

The result is imperfect in places — and that's the record. It's an honest snapshot of what I could build unaided at that point in 2020. The contrast with what I can build now, with a full toolkit and AI assistance, is the whole story of how I've grown as a developer.

---

## Design philosophy

Brutalist web design takes its cues from brutalist architecture: raw materials left exposed, structure as aesthetic, function over ornament. Where conventional web design hides the grid, brutalism shows it. Where conventional design smooths every edge, brutalism leaves them sharp.

It's not unfinished — it's honest.

---

## Tech stack

- HTML5
- CSS3
- No frameworks, no libraries, no references

---

## Context

Built in 2020 alongside other projects during a Certificate in Information Technology. Written as a personal creative exercise rather than a class requirement — demonstrating that the interest in building things went beyond what was being assessed.
